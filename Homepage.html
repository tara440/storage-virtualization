How storage virtualisation works

storage virtualisation serves as a in any given system at a point of time its of utmost importance since it enables the system to run effectively
and backs up its data that are needed for its functioning there are five types of storage virtualisation that is

  1. • Disk drive virtualization takes place when the firmware abstracts the physical cylinder, head, and sector into a single logical block address.
  2. • Storage system partitioning occurs when the storage system creates multiple partitions from a single storage resource.
  3. • Block virtualization happens when the software abstracts multiple disk arrays into a single storage resource.
  4. • Tape virtualization occurs as the software abstracts both tape drives and tape media onto a single disk resource.
  5. • File virtualization is the creation of an abstraction layer between the file servers and the clients who access those file servers.

Benefits of storage virtualisation

  1. Better Management
  2. Easy Retrieval and Upload of Data
  3. Security
  4. Improve agility and ability to scale
